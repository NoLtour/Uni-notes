#### What is
## Min drag in steady level flight
Using the equations from [[steady flight and true airspeed]]:

> $$D_{min} = 2\sqrt{A'B'}$$ 
>> where:
>> $D_{min}=$  Minimum drag
>> $A'=$ constant
>> $B'=$ constant


> $$D_{min} =  2\sqrt{   A'B'   }$$ 
>> where:
>> $D_{min}=$  Minimum drag
>> $A'=$ constant
>> $B'=$ constant

>> $A'=$ A constant $=\frac{1}{2}\rho S C_{Do}$
>> $B'=$ A constant $=\frac{2 K}{ \pi A} \cdot \frac{ W^{2}}{ \rho S }$
