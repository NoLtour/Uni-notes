#### What is
## Min drag speed in steady level flight
Using the equations from [[steady flight and true airspeed]]:

> $$V_{MD} = \sqrt[4]\frac{B'}{A'}$$ 
>> where:
>> $V_{MD}=$  Minimum drag speed
>> $A'=$ constant
>> $B'=$ constant
>> (They are only constants in steady flight)
