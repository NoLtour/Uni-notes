## Valence
The valence of an element is a measure of it's capacity to combine with other atoms when forming compounds or molecules. Ut us measured by the number of hydrogen atoms it combines with, for example:
| Element  | Hydrogen compund | Valence |
| -------- | ---------------- | ------- |
| Hydrogen | $H_2$            | 1       |
| Carbon   | $CH_4$           | 4       |
|          |                  |         |
