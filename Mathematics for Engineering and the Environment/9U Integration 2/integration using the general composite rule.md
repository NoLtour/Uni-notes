#### What is
## Integration using the general composite rule

> ### $$ \int f^{'}(g(x))\times g^{'}(x)\cdot dx = f(g(x))+k $$ 
>> where:
>> $f(x),g(x)=$ a function 
>> $k=$ a constant

### Examples
#### Example 1
> Find the [[Indefinate integral]] of $\int 2x\sqrt{x^{2}+2}\cdot dx$
