## Operator notation to define linear equations

Any equation that can satisfy the following is a [[linear differential equation]]:

> ### $$ L[ax_1 + bx_2] = aL[x_1] + bL[x_2] $$ 
>> where:
>> $x_1,x_2=$ all functions of x 
>> $a,b=$ any constant
>> $L[...]=$ a function function
