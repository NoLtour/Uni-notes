## Operator notation to define linear equations

Any equation that can satisfy the following is a linear equation:

> ### $$ L[af_1(x) + bf_2(x)] = aL[f_1(x)] + bL[f_2(x)] $$ 
>> where:
>> $f_1(x),f_2(x)=$ all functions of x 
>> $a,b=$ any constant
>> $L[]=$ a function
